Ufx-Link-Include Example

This example shows how the ufx-link-include can be used for a web site that loads parts of the document dynamically, e.g. for craeting a single page application (SPA) like this one.

The application has one static page

  • Index.html
and four parts that are loded dynamically when the users clicks the links in the navigation bar or in the footer.
  • home.html
  • Products.html
  • About.html
  • Contact.html
